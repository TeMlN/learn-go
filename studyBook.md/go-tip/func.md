# func in go



> ``` close : 채널을 닫을 떄 사용```

>``` len : 문자열, 배열, 슬라이스, 맵, 채널의 요소 개수 확인```

>``` cap : 문자열, 배열, 슬라이스, 맵, 채널의 최대 용량 확인```

>``` new : 구조체를 위한 메모리를 생성할 때 사용 ```

>``` make : 참조 타입(슬라이스, 맵, 채널)을 위한 메모리를 생성할 때 사용```

>``` copy : 배열 또는 슬라이스 복사```

>``` append : 슬라이스에 요소 추가```

>``` panic, recover : 에러 처리에 사용```

>``` complex, real, imag : 복소수 처리에 사용```
